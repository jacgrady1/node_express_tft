# node_express_tft

1. Clone the project
   
   ```sh
   git clone https://github.com/jacgrady1/node_express_tft.git 
    ```
2. cd into node_express_tft folder && Intall Dependencies

    ```sh
   npm install
    ```
    
3. run the project
   ```sh
   npm run dev
    ```
4. go to your browser and type in
   
   ```sh
   http://127.0.0.1:3000
    ```
